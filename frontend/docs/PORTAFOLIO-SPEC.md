# Spec: Portafolio Fullstack Developer

## Proyecto
Portafolio personal para desarrollador fullstack (single page, scroll)

## Estilo
- **Theme:** Dark mode técnico / cyber
- **Background:** `#0a0a0f` (casi negro)
- **Texto:** Blanco/gris claro (`#e5e5e5`, `#a3a3a3`)
- **Acento:** Púrpura cyber (`#8b5cf6`, `#a78bfa`)
- **Tipografía:** Sans-serif técnica (Inter), mono para código

## Secciones (scroll vertical)
1. **Hero** — Nombre, rol, CTA animado (typewriter/glitch effect)
2. **Sobre mí** — Introducción breve, experiencia
3. **Tech Stack** — Grid de tecnologías
4. **Proyectos** — Cards con demo/repo links
5. **Contacto** — Email + redes sociales

## Efectos
- Fade-in al scroll
- Glow sutil en acentos
- Terminal-style decoraciones

## Tech
React 18 + Vite + Tailwind CSS
